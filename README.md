# yl-template-node

Reusable YardaLab Node.js / TypeScript template repository for starting YardaLab-compatible projects.

## Purpose

This repository provides the initial foundation for future YardaLab Node.js / TypeScript projects.

Its goal is to become a clean template that helps new projects start with consistent repository structure, documentation, tooling, validation, and development workflow.

## Current status

This repository is currently in the initial structure and setup phase.

At this stage, it contains only the base repository files and placeholder directories needed to establish the project layout.

The following areas are intentionally not implemented yet:

* npm package setup
* TypeScript configuration
* YardaLab tooling integration
* commit message hook setup
* CI configuration
* test setup
* release or publishing workflow

These parts will be added by follow-up tasks.

## Repository layout

```text
yl-template-node/
  docs/
  src/
  tests/
  .gitignore
  LICENSE
  README.md
```

The `docs/`, `src/`, and `tests/` directories are currently kept as placeholders so the initial project structure is visible and tracked by Git.
